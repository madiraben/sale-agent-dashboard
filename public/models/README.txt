FREE 3D MODEL SOURCES FOR AI ROBOT/BOT:

1. SKETCHFAB (https://sketchfab.com)
   - Search: "robot", "AI bot", "assistant bot"
   - Filter: Free Download, CC Attribution
   - Format: Download as .glb or .gltf
   - Popular free models:
     * Search "Low Poly Robot" - many free options
     * Search "AI Assistant" - modern bot designs

2. POLY HAVEN (https://polyhaven.com/models)
   - All models are free (CC0 license)
   - Search for robot/character models
   - Download as .glb format

3. FREE3D (https://free3d.com)
   - Search: "robot", "bot", "AI assistant"
   - Filter by: Free models
   - Download .glb or .gltf format

4. TURBOSQUID (https://www.turbosquid.com)
   - Go to "Free 3D Models" section
   - Search: "robot", "AI bot"
   - Many high-quality free models available

5. CGTRADER (https://www.cgtrader.com)
   - Filter by: Free models
   - Search: "robot", "AI assistant"
   - Download .glb format

SETUP INSTRUCTIONS:
1. Download a .glb or .gltf file from any source above
2. Rename it to "robot.glb" (or update the path in AiAnimated.tsx)
3. Place it in: public/models/robot.glb
4. The component will automatically load and display it

RECOMMENDED MODELS:
- Low poly style works best for web performance
- File size: Under 5MB recommended
- Format: .glb preferred (smaller file size)

ALTERNATIVE: If you want to use a different model path, update line 99 in:
src/components/leadingpage/AiAnimated.tsx
